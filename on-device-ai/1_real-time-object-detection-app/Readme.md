
# On-Device AI Introduction: Real-Time Object Detection App

**"Give your React Native apps a brain, not just a cloud connection."**

Build the next generation of mobile apps with real-time, serverless intelligence.
Have you ever wanted to build apps that "see" and "understand" the world in real-time? Usually, this requires expensive cloud GPUs, complex APIs, and high latency. Not anymore.
With the arrival of ExecuTorch, Meta’s cutting-edge on-device AI engine, you can now run industry-standard PyTorch models directly on a user’s phone. No internet required. Zero latency. Total privacy.

## Why join this course?
In this micro-course, we strip away the academic jargon and get straight to building. You’ll go from "What is a tensor?" to deploying a Real-Time Object Detection app that identifies objects at lightning speed using your phone's camera.

* Native Performance: Leverage the power of C++ and the New Architecture in React Native.
* Production Ready: Learn the workflow used by top-tier engineering teams at Meta and Software Mansion.
* Portfolio Gold: Build a high-impact AI project that stands out in the crowded React Native job market.

## 📚 Course Curriculum

### **Module 1: The AI Paradigm Shift**
* **Ch 1: The ExecuTorch Revolution:** Understanding the jump from PyTorch Mobile to ExecuTorch. Why the New Architecture in React Native makes this the perfect time to learn this.
* **Ch 2: How AI "Sees":** A deep dive into Object Detection. We’ll explain Tensors, Bounding Boxes, and Confidence Scores in a way that actually makes sense to a JavaScript developer.

### **Module 2: The Native Bridge & Environment**
* **Ch 3: Architecture Deep-Dive:** How the C++ layer communicates with TypeScript. We'll look at how Software Mansion’s library acts as the bridge.
* **Ch 4: The .pte Filesystem:** What is a `.pte` file? How models are shrunk down and "serialized" to fit inside a mobile app bundle without killing performance.

### **Module 3: The Hands-on Build (The Core)**
* **Ch 5: Setting Up the Lab:** Initializing the project with the New Architecture enabled. Installing the necessary Turbo Modules.
* **Ch 6: The Vision Camera Feed:** Using `react-native-vision-camera` to tap into the raw hardware stream.
* **Ch 7: Wiring the Inference Hook:** Connecting the camera frames to the ExecuTorch engine using `useObjectDetection`. This is where the magic happens.
* **Ch 8: High-Performance UI:** Drawing bounding boxes that don't lag. Learn to use Skia or Reanimated to ensure the UI stays buttery smooth at 60 FPS.

### **Module 4: Graduation & Real-World Deployment**
* **Ch 9: iOS vs. Android:** Handling the quirks of different mobile GPUs (NPU vs. DSP).
* **Ch 10: Beyond Detection:** A look at the future—how to swap the model for OCR (text reading) or Image Segmentation.
